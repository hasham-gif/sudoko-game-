# 🎮 Sudoku Game — Java Swing

A classic **Sudoku puzzle game** built with Java Swing featuring a graphical user interface, number selection, error tracking, and win detection.

---

## 📸 About the Project

This is a fully functional Sudoku game where the player fills in the missing numbers on a 9x9 grid. The game tracks errors and congratulates the player upon successful completion.

---

## ✨ Features

- 🎯 **Interactive 9x9 Sudoku Board** — Click tiles and fill in numbers
- 🔢 **Number Selector Panel** — Choose numbers 1–9 from the bottom panel
- ❌ **Error Tracking** — Counts wrong answers and displays them
- ✅ **Win Detection** — Shows a congratulations message when puzzle is complete
- 🎨 **Visual Grid Borders** — 3x3 box separators clearly highlighted
- 🖼️ **Custom App Icon** — Personalized window icon

---

## 🛠️ Built With

- **Java** — Core programming language
- **Java Swing** — GUI framework
- **AWT** — For colors, fonts, and borders

## 🎮 How to Play

1. Launch the game
2. Click a number (1–9) from the bottom panel
3. Click an empty tile on the board
4. Correct answer ✅ stays on board
5. Wrong answer ❌ increases error count
6. Fill all tiles to WIN! 🎉
